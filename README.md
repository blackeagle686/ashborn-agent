# 🐦‍🔥 Ashborn Agent

**The Ultimate Autonomous Architect Powered by Phoenix AI**

Ashborn is a state-of-the-art autonomous agent designed to manifest complex visions into production-ready code. Built on the high-performance **Phoenix AI Framework**, Ashborn leverages a parallel cognition cycle to think, plan, act, and reflect with unprecedented efficiency.

---

## ✨ Features

- **🧠 Advanced Cognition**: Parallel Awareness (Think + Analyze) for deep understanding of user intent and workspace context.
- **🛠️ Autonomous Planning**: Multi-step planning with a self-correcting Reflection loop.
- **⚡ Parallel Execution**: Execute multiple tools simultaneously to speed up complex workflows.
- **📺 Premium Streaming UI**: A beautiful CLI experience with real-time progress bars and token-by-token streaming Markdown output.
- **💾 Hybrid Memory**: Long-term and short-term memory systems powered by vector databases and semantic search.
- **🏗️ Project Scaffolding**: Instantly generate production-ready microservices, CLIs, and more.

## 🚀 Getting Started

### Prerequisites
- Python 3.10+
- [Phoenix AI Framework](https://github.com/blackeagle686/phoenix-ai)
- OpenAI API Key (or a compatible provider like LongCat AI)

### Installation
1. **Clone the repository:**
   ```bash
   git clone https://github.com/your-username/ashborn-agent.git
   cd ashborn-agent
   ```

2. **Set up environment variables:**
   Create a `.env` file in the root directory:
   ```env
   OPENAI_API_KEY=your_api_key_here
   LOG_LEVEL=WARNING
   ```

3. **Install dependencies:**
   ```bash
   pip install -r requirements.txt
   ```

## 🛠️ Usage

Ashborn comes with a feature-rich CLI for both interactive and direct task execution.

### Interactive Mode
Start a deep conversation with the agent:
```bash
python main.py interact
```

### Direct Project Generation
Quickly scaffold a new project:
```bash
python main.py generate "my-awesome-api" "python_microservice"
```

## 🏗️ Architecture

Ashborn is built exclusively on the **Phoenix AI Framework**, utilizing its core modules:
- **`phoenix.agent`**: Orchestrates the autonomous loop.
- **`phoenix.cognition`**: Powers the Thinker, Planner, and Reflector.
- **`phoenix.llm`**: High-performance streaming interface for LLM providers.
- **`phoenix.memory`**: Advanced semantic memory management.

For details on recent framework optimizations made for Ashborn, see [phoneix-updates-notes.md](./phoneix-updates-notes.md).
