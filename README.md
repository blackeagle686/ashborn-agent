## 🚀 Standalone Installation

Ashborn is now easier than ever to install on Linux.

1. **Download the Bundle**: Get the latest `ashborn-ide-linux.tar.gz` from your server.
2. **Extract and Run**:
   ```bash
   tar -xzf ashborn-ide-linux.tar.gz
   bash install.sh
   ```
3. **Launch**: Find **Ashborn IDE** in your application menu or run `~/.ashborn/ashborn-ide` from the terminal.

> **Note**: The installer automatically handles VSCodium download, Python virtual environments, and Desktop integration.

## 📦 Distribution & Packaging

Are you a developer looking to share Ashborn with your team? Use our built-in packaging system.

### Create a Distribution Package
Run the packaging script to bundle the extension, backend, and installer:
```bash
bash scripts/package.sh
```
This generates a production-ready archive in `dist/ashborn-ide-linux.tar.gz` ready for server upload.
